Build Hexo Blog

Posted on 2019-10-05 Edited on 2019-10-08

Step1

Download node.js:
下载地址
打开终端，切换到root用户: sudo su
查看node.js版本: node -v
npm -v
(如果没有梯子的话，可以使用阿里的国内镜像进行加速。
npm config set registry https://registry.npm.taobao.org)
install hexo:
npm install -g hexo-cli
查看hexo版本: hexo -v

Step2

新建一个myblog文件夹: mkdir myblog
cd myblog
初始化博客: sudo hexo init
启动博客: hexo s
ctrl+c断开以后，建立一篇新的文章: hexo n "My First Blog"
cd source/_posts/ 进入posts
将Hexo与GitHub关联起来，打开配置文件_config.yml
deploy:
type:git
repo:输入GitHub创建仓库的路径
branch:master

Step3

hexo clean 并不一定需要
hexo g
hexo s
